# csv2md

csv converter to md file

## SYNOPSIS
    Usage: csv2md [OPTIONS] < input.csv > output.md

## DESCRIPTION
    Convert a csv file with ; separator to a markdown table.

    --set-title
      add a title in the markdown file
      
    --version
      display version and quit

    --help
      display help and quit

    Exit status:
        0      if OK,
        1      if any problem
